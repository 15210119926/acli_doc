## 1. 概述

本仓库集中管理 `acli` 命令体系的全部命令说明文档与官方使用示例等，提供：

- 命令行使用手册
- 典型排障案例
- ...

## 2. 快速参与

### 2.1 前置要求

- Node.js ≥ 18
- 熟悉 Markdown 语法（文档写作）

### 2.2 文档修改流程

1. 定位待改文档
2. 本地修改并保存

### 2.3 本地部署及测试

修改并保存完毕后，您可以尝试在本地部署或测试，操作流程如下：

```bash
npm install          # 安装项目依赖

# 本次测试
npm run start        # 启动本地开发服务器，带有热重载功能
# 访问 http://localhost:3000 查看实时效果

# 本地部署
# npm run build      # 构建生产版本的静态网站
# npm run serve      # 在本地启动服务器预览构建后的网站
```
