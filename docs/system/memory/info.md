# info
操作概述: 查看主机内存的使用信息

命令参数:
```bash
无
```

使用示例:
```bash
acli system memory info
```

结果示例:
```bash
MemTotal:       40679596 kB
MemFree:        20305748 kB
MemAvailable:   24797012 kB
Buffers:          912428 kB
Cached:          5285744 kB
SwapCached:            0 kB
Active:         12311456 kB
```
