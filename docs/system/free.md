# free
操作概述: 调用系统free命令

命令参数:
```bash
无
```

使用示例:
```bash
acli system free
```

结果示例:
```bash
# acli system free -k
             total       used       free     shared    buffers     cached
Mem:      40675504   21955264   18720240          0     832328    5458348
-/+ buffers/cache:   15664588   25010916
Swap:     33554428          0   33554428
```
