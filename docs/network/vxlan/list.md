# list
操作概述: 查询集群vxlan信息列表

命令参数:  
```bash
无
```

使用示例:
```bash
acli network vxlan list
```

结果示例:
```bash
   vxlan_clusters:
      probe_enable: 1
      probe_latency_us: 10000
      probe_timeout_ms: 3000
      id: 8a5987e4-ff3e-47d0-8fd1-b36d2d566439
      probe_interval_s: 30
      jumbo_frame: 1
      port: 8472
```
